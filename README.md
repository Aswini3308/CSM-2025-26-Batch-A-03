# 🌐Indicator Engineering on a Cloud-Edge GIS Platform Assisting Post-Seismic Reconstruction and Rehabilitation


## Overview
This project presents a GIS-based platform integrated with cloud–edge computing to support post-seismic reconstruction monitoring. The system tracks reconstruction activities across 56 municipalities affected by the 2009 earthquake in Italy.

The platform provides real-time access to geospatial data, enables project tracking, and supports decision-making for reconstruction management. By using a hybrid cloud-edge architecture, the system improves data processing efficiency, scalability, and transparency.

## Features
- GIS-based visualization of reconstruction data
- Cloud and edge computing integration
- Real-time monitoring of reconstruction progress
- Project tracking across municipalities
- Decision support for stakeholders

## Keywords
GIS, Cloud Computing, Edge Computing, Post-Seismic Reconstruction, Hybrid Architecture, Real-Time Data
